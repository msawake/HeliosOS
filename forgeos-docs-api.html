Health & System

GET/api/healthPublic
System health check. Returns component status, agent count, pending approvals.
{ "status": "ok", "components": { "database": true, "agents_registered": 41, "llm_providers": ["anthropic"] } }
GET/api/readinessPublic
K8s readiness probe. Returns 503 until boot is complete.
GET/metricsPublic
Prometheus scrape endpoint. Returns metrics in exposition format.

Agents

GET/api/platform/overviewPublic
Registry summary: counts by stack, execution type, and status.
GET/api/platform/agentsPublic
List all agents. Filter: ?stack=forgeos&execution_type=scheduled&department=sales
GET/api/platform/agents/{agent_id}Public
Get a single agent definition with status.
POST/api/platform/agentsAuth Required
Deploy a new agent. Returns 201 with agent definition.
{
  "name": "my-agent",
  "stack": "forgeos",
  "execution_type": "reflex",
  "description": "A helpful assistant",
  "chat_model": "claude-sonnet-4-5-20250514",
  "tools": ["mcp__filesystem__*"],
  "system_prompt": "You are a helpful assistant."
}
PUT/api/platform/agents/{agent_id}Auth Required
Update an existing agent (tools, system prompt, LLM config, execution type).
POST/api/platform/agents/{agent_id}/invokeAuth Required
One-shot agent invocation. Returns AgentResult with output text and tool calls.
{ "prompt": "Generate the daily report", "context": {} }
POST/api/platform/agents/{agent_id}/chat/streamPublic
Streaming multi-turn chat via SSE. Events: text_delta, tool_call, tool_result, hitl_request, done, error.
{ "message": "Hello!", "session_id": "optional-session-id" }
GET/api/platform/agents/{agent_id}/chat/sessionsPublic
List chat sessions for an agent.
GET/api/platform/agents/{agent_id}/chat/historyPublic
Get full message history for a session. Query: ?session_id=...
DELETE/api/platform/agents/{agent_id}/chat/sessions/{session_id}Public
Delete a chat session.
POST/api/platform/agents/{agent_id}/stopAuth Required
Stop a running agent (cancels loops, removes scheduled jobs).
DELETE/api/platform/agents/{agent_id}Auth Required
Undeploy and delete an agent (stops, removes files, unregisters).

Agent Wizard

POST/api/platform/wizard/chatAuth Required
AI-assisted agent design. Send conversation messages, get agent config proposals or follow-up questions.
{ "messages": [{ "role": "user", "content": "Create a sales lead scoring agent" }], "context": {} }

Events

GET/api/eventsPublic
List events. Filter: ?department=sales&status=PENDING&priority=P0_CRITICAL
POST/api/platform/eventsAuth Required
Fire a custom event to the event bus.
{ "name": "lead.qualified", "payload": { "lead_id": "L-123" } }

Approvals (HITL)

GET/api/approvalsPublic
List all pending approval requests. Filter: ?category=financial
GET/api/approvals/{request_id}Public
Get approval request details and status.
POST/api/approvals/{request_id}/approvePublic
Approve a pending request. Body: { "approved_by": "user@example.com", "notes": "..." }
POST/api/approvals/{request_id}/rejectPublic
Reject a pending request. Body: { "rejected_by": "user@example.com", "reason": "..." }

Admin Chat

POST/api/admin/chatPublic
Non-streaming admin chat. Supports commands: "list agents", "system status", "show approvals".
POST/api/admin/chat/streamPublic
Streaming admin chat via SSE. Same commands plus full LLM conversation.
GET/api/admin/healthPublic
Detailed system health with component breakdown.
GET/api/admin/metricsPublic
Admin dashboard metrics (agent counts, event counts, etc.).
GET/api/admin/eventsPublic
List recent events for admin dashboard.
GET/api/admin/knowledgePublic
Search/list knowledge base entries.

Intelligence

POST/api/intelligence/askPublic
Ask the intelligence system a question (ontology-aware).
GET/api/intelligence/ontology/schemaPublic
Get the ontology schema (entity types, relationships).
GET/api/intelligence/ontology/objectsPublic
List ontology objects. Filter: ?type=Lead&limit=50
POST/api/intelligence/connectors/syncAuth Required
Trigger a data connector sync (pulls data from external systems).

Audit

GET/api/auditAuth Required
Query audit log. Filter: ?action=agent.deploy&limit=100

Billing & Usage

GET/api/billing/usageAuth Required
Current billing period usage (tokens, cost, invocations).

Providers

GET/api/admin/providersPublic
LLM provider status (which are configured, which are active).

Auth

POST/api/auth/tokenPublic
Generate a dev API token. Body: { "name": "dev-token" }
GET/api/meAuth Required
Get current user/token info.

Clients

POST/api/clientsAuth Required
Create a new client.
GET/api/clientsAuth Required
List all clients.
GET/api/clients/{client_id}Auth Required
Get client details.
DELETE/api/clients/{client_id}Auth Required
Delete a client.
GET/api/clients/{client_id}/agentsAuth Required
List agents owned by a client.
POST/api/clients/{client_id}/mcp-serversAuth Required
Add a client-scoped MCP server configuration.
GET/api/clients/{client_id}/mcp-serversAuth Required
List MCP servers configured for a client.
DELETE/api/clients/{client_id}/mcp-servers/{server_name}Auth Required
Remove a client MCP server.

GET/api/workflows/{workflow_id}Auth Required
