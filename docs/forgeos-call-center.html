Agent uses human__ask (structured question with options + deadline) or human__notify (one-way alert). Delivered via dashboard, Slack, or email based on priority.

A2H
▶

Trigger: Customer anger spike detected on Maria's call #4521. Sentiment score dropped from 0.6 to 0.15 in 30 seconds.

Sentiment Monitor→human__ask(priority=P0_CRITICAL)→Rachel (dashboard + slack)
human__ask(
  question: "Call #4521 — customer anger spike. Maria may need help."
  response_type: "choice"
  options: [Take Over, Coach CSR, Monitor Only]

Denied Tool Examples
What happens when an agent tries a denied tool:
Call Router tries crm_update:
  Stage 2 [capability] → PermissionManager: "crm_update" in denied list
  SHORT CIRCUIT → deny("Tool 'crm_update' is explicitly denied")
  Why: Routing agent can READ customer data but must NOT modify records

Sentiment Monitor tries search_knowledge:
  Stage 2 [capability] → "search_knowledge" in denied list
  deny
  Why: Sentiment analysis is real-time signal processing — it should NOT query the knowledge base (different concern, different budget)

Quality Scorer tries publish_event:
  Stage 2 [capability] → "publish_event" in denied list
  deny
  Why: Quality scores are internal — they go through add_decision (audited, reviewable), not broadcast events

Live Scenario — 60 Seconds in the Call Center
What happens in a single minute when multiple calls are active simultaneously.
T+0s — Incoming call from Jane Smith (premium customer)
1. Call Router detects incoming call → agent__call → Customer Profiler
2. Profiler pulls history: premium tier, declining sentiment, last issue was billing
3. Router decision: route to Maria (billing specialist + previous relationship)
Kernel: 4 tool calls gated, 1 A2A call (cross-namespace with token), 2 budget checks
T+15s — Maria picks up the call
4. Maria sees customer briefing card (pre-populated by Profiler)
5. Sentiment Monitor begins tracking call #4521
6. Maria asks Knowledge Assistant: "refund policy for premium?" → H2A
7. Knowledge Assistant returns answer in 1.2s
Kernel: 1 H2A call, 1 tool call (search_knowledge), budget: $0.08
T+35s — Customer gets angry about billing charge
8. Sentiment Monitor detects anger spike (0.6 → 0.15)
9. Sentiment Monitor → A2A → Escalation Manager (cross-namespace token)
10. Escalation Manager → A2H → Rachel (P0_CRITICAL: "Maria needs help")
Kernel: 1 A2A (token issued + revoked), 1 A2H (human__ask), 2 tool calls, 1 audit event
T+50s — Rachel responds via Slack
11. Rachel clicks "Coach CSR" on Slack interactive message
12. H2A response recorded: {choice: "Coach CSR"}
13. Escalation Manager sends de-escalation script to Maria's dashboard
Kernel: 1 A2H response recorded, 1 notification delivered, audit logged
T+60s — Call continues with coaching support
14. Maria uses de-escalation script, customer calms down
15. Sentiment Monitor detects recovery (0.15 → 0.55)
16. Meanwhile: Call Router processes 2 more incoming calls, Customer Profiler runs 2 more lookups
Total kernel decisions in 60 seconds: ~25
Protocols used: H2A (1), A2H (1), A2A (3), Syscall (~18 tool calls)
